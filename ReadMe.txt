This is an endless platformer game where the character constantly bounces, goal is to get the highest score.

Press Left or Right buttons to move the character.
Press "R" to restart the game once the game is over.

NOTE:
The RetroGaming.ttf font inside the Assets folder needs to be installed for the game to work.
